1. What is the DOM in JavaScript?

The DOM (Document Object Model) is a structured representation of the HTML or XML document. It allows JavaScript to access and manipulate the content, structure, and style of a webpage dynamically.

2. How can you select an element from the DOM using JavaScript?

You can select an element from the DOM using methods like getElementById(), querySelector(), getElementsByClassName(), or getElementsByTagName().

3. How do you create and insert a new element into the DOM?

To create and insert a new element into the DOM, you use createElement() to create the element, modify it as needed, and then insert it using methods like appendChild() or insertBefore().

4. How can you remove an element from the DOM?

You can remove an element from the DOM using the remove() method on the element, or by calling the removeChild() method on the parent of the element.
